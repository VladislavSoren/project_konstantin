# Программа для учета автомобилей и клиентов в автосервисе

## Функционал сервиса:
- ввод, хранение и предоставление информации о клиентах автосервиса
- ввод, хранение и предоставление информации об автомобилях
- ввод, хранение и предоставление документов о ремонтах автомобилей (выполненных работах и замененных запчастях)

## Архитектура:
- Frontend (Django)
- Backend (Django)
- БД (PostgresSQL)
- 
## Проектная реализация:
- Django поднята в Docker контейнере на продуктовом сервере
- БД (PostgresSQL) поднята в Docker контейнере на продуктовом сервере
- Миграции (alembic) # что это такое?
